## 简介

本仓库提供了一套基于 ElasticSearch(搜索) + Cassandra(存储) 的整套海量数据存储方案，基于 ABP vNext 重写了仓储层实现，减少上层业务代码的侵入。具体存储架构与网络拓扑信息，请参考下面 **架构** 一节。
